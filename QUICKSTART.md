## Super Simple Setup

### 1. Install VRChat Creator Companion (2 minutes)
1. Download VCC: https://vrchat.com/home/download
2. Install and launch VCC
3. Sign in with your VRChat account

### 2. Open This Project (1 minute)
1. In VCC, click **"Add"** ‚Üí **"Add Existing Project"**
2. Select this folder: `VRCHAT-COIN_Operated`
3. VCC will auto-install all dependencies
4. Click **"Open Project"** to launch Unity

### 3. Test in VRChat (2 minutes)
1. In Unity, go to: **VRChat SDK ‚Üí Show Control Panel**
2. Click the **"Builder"** tab
3. Click **"Build & Test"**
4. Put on your VR headset and test!

**That's it!** You now have a working VRChat world.

## Troubleshooting

**VCC won't recognize the project?**
- Make sure `vpm-manifest.json` exists in the root folder
- Try: VCC ‚Üí Settings ‚Üí Clear Cache

**Unity won't open?**
- Install Unity 2022.3.22f1 via VCC
- VCC ‚Üí Settings ‚Üí Unity Editors ‚Üí Install

**Build errors?**
- Wait for Unity to finish importing (first time takes ~5 min)
- Check Unity Console for specific errors
